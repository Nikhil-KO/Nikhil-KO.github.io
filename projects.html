Computer vision based head tracking

A free open source head tracking solution using only image data from webcam. Image processing using OpenCV and Dlib for facial landmark model. GUI built using .NET and WPF. The resulting data is send to OpenTrack via a UDP socket, which is then translated to head movement in game.
